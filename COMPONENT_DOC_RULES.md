# COMPONENT_DOC_RULES.md

Every generated component must include documentation.

Documentation format:

Component Name

Purpose

Props

Example usage

API data used

Dependencies

Example:

Component: NewsCard

Purpose:
Displays a single news item.

Props:
title
image
date
description

Example usage:

<NewsCard
  title="Tech Fest"
  date="12 March"
/>

Dependencies:
Card
ReadMoreButton
